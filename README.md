# 📦 QR Code & Password Generator

Projeto simples em **Node.js** que permite gerar um **QR Code** ou uma **senha aleatória**, diretamente pelo terminal.

---

## 🚀 Funcionalidades

- 📱 Gerar QR Code a partir de um link  
- 🔐 Gerar senha aleatória  

---

## 🛠️ Tecnologias

- JavaScript  
- Node.js  
- Pacotes NPM:  
  - prompt  
  - chalk  
  - qrcode-terminal  

---

## 📁 Estrutura

```
src/
├── index.js
├── prompts/
├── services/
│   ├── qrcode/
│   └── password/
```

---

## 💻 Uso

Ao executar o projeto, o usuário escolhe no terminal:

- **1** → Gerar QR Code  
- **2** → Gerar senha  

---

## 📄 Descrição

O projeto foi estruturado separando cada funcionalidade em suas próprias pastas e arquivos, facilitando organização e manutenção do código.
